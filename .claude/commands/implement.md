# Implement Feature Workflow

## Usage
```
/implement <feature description>
```

## Workflow

### 1. Research Phase
Use `project-research-analyst` agent

### 2. Implementation
- Follow offline-first pattern
- Use Redux for state
- Use StyleSheet for styles

### 3. Quality Assurance (Parallel)
- `code-refactoring-optimizer`
- `security-style-reviewer`
- `test-coverage-validator`

### 4. Testing
- Test on Android emulator
- Test offline behavior
- Test sync functionality

### 5. Documentation
Use `documentation-agent`
